# FirstWitness

FirstWitness is an AI-powered emergency incident reporting platform that enables users to quickly report crimes, accidents, and emergencies with location, images, and guided assistance.

## Features

- 📍 Live location detection
- 📸 Capture and upload incident photos
- 🚨 Emergency contact information
- 🤖 AI-guided incident reporting
- 📝 Generate structured reports
- 🌐 Google Places integration

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- Python

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
